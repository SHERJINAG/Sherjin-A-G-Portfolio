## 💻 Local Development

   ```bash
   git clone https://github.com/sherjinag/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```



3. **Start the development server**:
   ```bash
   npm run dev
   ```
